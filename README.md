*Fortune 500 Financial Performance Dashboard
An end-to-end data analytics project leveraging Alteryx for data preparation and Power BI for interactive business intelligence reporting. This dashboard analyzes Fortune 100 company financial performance, profitability, and ranking changes through dynamic visualizations and cross-filtering functionality.

*Project Overview
This project transforms raw Fortune 500 financial data into actionable business insights by:

Cleaning and transforming financial datasets in Alteryx
Calculating key performance metrics such as Profit Margin and Rank Change
Categorizing companies by profitability levels
Building an interactive Power BI dashboard with cross-filtering and drillthrough functionality
Visualizing company performance, profitability, and ranking movements

*Key Features
Data Preparation (Alteryx)
Data cleansing and validation
Currency and numeric field standardization
Profit Margin calculations
Profitability category classification
Rank Change calculations

*Power BI Dashboard

Revenue vs. Profit scatter plot analysis
Interactive KPI cards for Revenue, Profit, and Profit Margin
Top Revenue Companies visualization
Profitability segmentation analysis
Rank movement tracking
Top 10 Rank Improvers and Decliners
Company-level drillthrough analysis
Dynamic filtering and cross-highlighting across visualizations
Conditional formatting for positive and negative rank movement
